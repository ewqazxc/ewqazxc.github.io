React 源码阅读记录

–20200730

用了N久的React，从未阅读过其源码，这次能看多少呢?

下载源码

从https://github.com/facebook/react.git直接下载了个zip压缩包代码，从目录上看。。。看不出怎么生成React的
在packages 下看到了react文件夹，这个看着更像是React源码。
然而，除了在src/React.js中看到一堆熟悉的Api，并不懂这东西是如何编译的，都不知道从哪里看起。。。
因此，还是参考下他人的学习记录吧！https://blog.csdn.net/qq_32281471/article/details/94968617

  1. 克隆代码到本地

  2. 查看分支git branch -a，切换到指定版本分支

    项目中都使用到16.9.3，但在分支中最高只到16.8.6 就先看这个吧

  3. 安转依赖yarn install ，从源码编译出 react.development.js 和 react-dom.development.js

    构建出的文件，在 build/node_modukes/react ，有 commonjs 和 umd 两个版本

  4. 直接打开 fixtures/packaging/babel-standalone/dev.html ，该页面使用的是最新编译出来的 react.development.js 和 react-dom.development.js
    此时，可以在 packages/react/src 下修改源码后，重新编译查看修改后的效果
